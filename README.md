🚪 Rapid Door Estimator

Rapid Door Estimator is a machine learning–driven configurator designed to support the REMAX sales team in generating fast, consistent, and data-driven door estimates.

The application predicts:

Door Sell Price
Door Cost
Margin (including reseller discount)
🚀 Key Features
🔹 1. Estimation Engine
Machine learning models trained on historical data (quotes, sales orders, jobs)
Predicts:
Sell Price
Cost
Automatically calculates:
Discounted Sell Price
Margin Value
Margin %
🔹 2. Configurator Interface
User-friendly UI aligned with existing M1 configurator
Sections include:
Door configuration (model, height, width)
Installation options
Curtain options
Dynamic DoorSellPrice lookup based on size and model
🔹 3. Pricing Breakdown (What-if Analysis)
Shows how selected options impact pricing
Helps answer questions like:
“How much does Traffic Light add to the price?”
“What is the price without Interlock?”

Provides estimated:

Price impact per feature
Cost impact per feature
🔹 4. Export to M1 Format
Generates Excel output compatible with M1 parameter structure
Includes:
Selected configurator values
Encoded inputs
Ready for integration workflows
🔹 5. SQL Integration (Search & Load)
Search historical:
Quotes
Load existing configurations into the estimator
Enables reuse and comparison of past jobs
